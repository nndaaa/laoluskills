## 关键设计决策

1. **永远不滚详情页** — HTML 字段都在初始 DOM 加载,滚动=触发风控
2. **永远用 HTML 属性提取**(`title` 属性)而不用 textContent — 属性稳定,文本会被插入图片
3. **detail page 主商品用独立类**(`desc--` / `price--` / `want--`),不与推荐位混用
4. **离线调试优先** — 抓一次存 HTML 到 dump/,然后调 extractor 离线改 100 次都行
5. **锚点定位 > 嵌套正则** — 找 `desc--` 位置然后往后扫到下一个 `labels--` 锚点,比 `<span>...</span>` 嵌套正则稳
6. **class 尾随空格** — 闲鱼有的 class 是 `class="price--OEWLbcxC "` (尾随空格),正则要用 `\s*` 容忍

## 风控经验

- **间隔 9s 是底线** — 4 关键词 4 秒内跑完会触发
- **不滚动详情页** — 触发风控信号(异常导航)
- **不要用 headless** — 必须 headful(Xvfb/ToDesk),因为要扫码
- **cookies 7-30 天有效** — 过期重扫
